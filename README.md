## Requirements
Before running the project, **make sure all dependencies below are installed and available in your system PATH**.
### Node.js
* Minimum version: **Node.js 16+**
* Download: [https://nodejs.org](https://nodejs.org)

### FFmpeg
Used for:
* Audio extraction
* Video/audio merging
* Format conversions

### yt-dlp (VERY IMPORTANT)
This is the **core engine** of the downloader.
If `yt-dlp` is not detected, the app **will not work**.

## Getting Started
### 1. Clone or Download the Project
```bash
git clone https://github.com/Asphyxiale/multiplatform-video-downloader.git
```
or download as ZIP and extract.

---
### 2. Open Terminal & Navigate to Folder
```bash
cd multiplatform-video-downloader
```
---
### 3. Install Node Dependencies
```bash
npm install
```
---
### 4. Start the Application
```bash
npm start
```
The downloader is now ready to use.
